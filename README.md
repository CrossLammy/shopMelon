## ภาพรวมของระบบ

ระบบร้านค้านี้เป็นโปรแกรม C++ แบบ Console Application สำหรับจำลองการซื้อสินค้าอิเล็กทรอนิกส์ ประกอบด้วยฟีเจอร์หลัก ดังนี้

- ดูรายการสินค้าในร้าน
- เพิ่ม / ลบสินค้าในตะกร้า
- ใช้รหัสส่วนลดและชำระเงิน
- พิมพ์ใบเสร็จ
- จัดการรหัสส่วนลดผ่านระบบแอดมิน

---

## การเริ่มต้นใช้งาน

### ข้อกำหนดเบื้องต้น
- Compiler: `g++` หรือ compiler ที่รองรับ C++11 ขึ้นไป
- ไฟล์ `discount_codes.txt` (สร้างอัตโนมัติเมื่อแอดมินสร้างรหัสส่วนลด)

### วิธีคอมไพล์และรัน

```bash
g++ -o shop main.cpp
./shop
```

## เมนูหลัก

เมื่อเปิดโปรแกรม จะแสดงเมนูดังนี้

```
================== SHOP MENU ==================
1. View product
2. Cart menu
3. Checkout
4. Admin menu
0. Exit
Select menu :
```

พิมพ์ตัวเลข `0–4` เพื่อเลือกเมนูที่ต้องการ

---

## 1. ดูสินค้า (View Product)

เลือก **`1`** จากเมนูหลัก

ระบบจะแสดงรายการสินค้าทั้งหมดในร้านพร้อม ID, ชื่อสินค้า และราคา

```
================== STORE INVENTORY ==================
ID: 1    | MacBook Air M2                | Price: 34900.00 Bath
ID: 2    | iPad Pro 11-inch              | Price: 32900.00 Bath
ID: 3    | Logitech MX Master 3S         | Price: 4390.00 Bath
...
=====================================================
```

> ใช้ **ID** ของสินค้าเพื่ออ้างอิงเมื่อต้องการเพิ่มสินค้าเข้าตะกร้า

---

## 2. ตะกร้าสินค้า (Cart Menu)

เลือก **`2`** จากเมนูหลัก จะแสดงรายการสินค้าและตะกร้าปัจจุบัน พร้อมเมนูย่อย

```
[1] add  [2] remove  [0] back
Select menu :
```

### เพิ่มสินค้า `[1]`

1. เลือก `1`
2. ป้อน **ID** ของสินค้าที่ต้องการ
3. ระบบจะยืนยันการเพิ่มสินค้า

```
>> Enter product ID: 3
>> Added --> [Logitech MX Master 3S] to cart!
```

> ⚠️ หากป้อน ID ที่ไม่มีในระบบ จะแสดงข้อความแจ้งเตือน

### ลบสินค้า `[2]`

1. เลือก `2`
2. ป้อน **ID** ของสินค้าที่ต้องการลบออกจากตะกร้า
3. ระบบจะยืนยันการลบสินค้า

```
>> Enter remove product : 3
>> remove --> [Logitech MX Master 3S] form cart!
```

> ⚠️ หากตะกร้าว่าง ระบบจะแจ้งว่าไม่มีสินค้าให้ลบ

### กลับเมนูหลัก `[0]`

พิมพ์ `0` เพื่อกลับสู่เมนูหลัก

---

## 3. ชำระเงิน (Checkout)

เลือก **`3`** จากเมนูหลัก

> ⚠️ ต้องมีสินค้าในตะกร้าก่อน หากตะกร้าว่างระบบจะแจ้งเตือนและกลับเมนูหลัก

### ขั้นตอนการชำระเงิน

**ขั้นที่ 1 — ระบบแสดงสรุปตะกร้า**
```
================== CART ==================
ID: 5 - Sony WH-1000XM5 (11900.00 Bath)
ID: 3 - Logitech MX Master 3S (4390.00 Bath)
==========================================
Total : 16290.00 Bath
```

**ขั้นที่ 2 — ป้อนรหัสส่วนลด (ถ้ามี)**
```
Enter discount code (0 to skip):
```
- หากมีรหัส ให้พิมพ์รหัสส่วนลด
- หากไม่มี พิมพ์ `0` เพื่อข้าม

ตัวอย่างเมื่อใช้รหัสส่วนลดสำเร็จ:
```
>> Discount code applied!
Discount : 10%
You save : 1629.00 Bath
>> Code has been used and removed!
Final price : 14661.00 Bath
```

> ⚠️ รหัสส่วนลดแต่ละรหัสสามารถใช้ได้ **ครั้งเดียว** เท่านั้น ระบบจะลบรหัสออกทันทีหลังใช้งาน

**ขั้นที่ 3 — ยืนยันการชำระเงิน**
```
Confirm Payment? (y/n):
```
- พิมพ์ `y` หรือ `Y` เพื่อยืนยัน → ระบบพิมพ์ใบเสร็จและล้างตะกร้า
- พิมพ์ `n` เพื่อยกเลิก → กลับสู่เมนูหลักโดยที่ตะกร้ายังคงอยู่

### ตัวอย่างใบเสร็จ

```
==========================================
              OFFICIAL RECEIPT
==========================================
Date: Sat Mar 07 12:00:00 2026
------------------------------------------
Product                         Price
------------------------------------------
Sony WH-1000XM5           11900.00 B
Logitech MX Master 3S      4390.00 B
------------------------------------------
Subtotal:                  16290.00 B
Discount:               -  1629.00 B
------------------------------------------
TOTAL TO PAY:              14661.00 B
==========================================
         THANK YOU FOR SHOPPING!
==========================================
```

## 4. เมนูแอดมิน (Admin Menu)

เลือก **`4`** จากเมนูหลัก จากนั้นเข้าสู่ระบบด้วย

| ช่อง | ค่าเริ่มต้น |
|---|---|
| Username | `admin` |
| Password | `1234` |

```
========== ADMIN LOGIN ==========
Username: admin
Password: 1234
>> Login successful! Welcome Admin.
```

เมื่อเข้าสู่ระบบสำเร็จ จะเห็นเมนูแอดมิน

```
[1] Random discount code
[2] Delete discount code
[3] View all discount codes
[0] Lockout
```

---

### [1] สร้างรหัสส่วนลด (Random Discount Code)

1. เลือก `1`
2. ป้อนเปอร์เซ็นต์ส่วนลด (1–100)
3. ระบบจะสร้างรหัสส่วนลดแบบสุ่มและบันทึกลงไฟล์

```
enter the discount percentage: 10

====== DISCOUNT CODE GENERATED ======
CODE: SAVE105423
Discount: 10%
=====================================
>> Saved to discount_codes.txt successfully!
```

> รหัสจะมีรูปแบบ `SAVE{เปอร์เซ็นต์}{ตัวเลขสุ่ม}` เช่น `SAVE105423`

---

### [2] ลบรหัสส่วนลด (Delete Discount Code)

1. เลือก `2`
2. ระบบแสดงรายการรหัสทั้งหมด
3. ป้อนหมายเลขลำดับที่ต้องการลบ (หรือ `0` เพื่อยกเลิก)
4. เลือกว่าจะลบเพิ่มเติมหรือกลับเมนู

```
====== DISCOUNT CODES LIST ======
[1] Code: SAVE105423 | Discount: 10%
[2] Code: SAVE2091   | Discount: 20%
=================================
Enter line number to delete (0 to cancel): 1
>> Discount code deleted successfully!
```

---

### [3] ดูรหัสส่วนลดทั้งหมด (View All Discount Codes)

1. เลือก `3`
2. ระบบแสดงรายการรหัสส่วนลดทั้งหมดที่มีอยู่

```
====== ALL DISCOUNT CODES ======
[1] Code: SAVE105423 | Discount: 10%
[2] Code: SAVE2091   | Discount: 20%
================================
```

3. พิมพ์ `0` เพื่อกลับเมนูแอดมิน

---

### [0] ออกจากระบบแอดมิน (Lockout)

เลือก `0` เพื่อออกจากโหมดแอดมินและกลับสู่เมนูหลัก

---

## ไฟล์ที่เกี่ยวข้อง

| ชื่อไฟล์ | คำอธิบาย |
|---|---|
| `main.cpp` | ซอร์สโค้ดหลักของโปรแกรม |
| `discount_codes.txt` | เก็บรหัสส่วนลดในรูปแบบ `CODE,เปอร์เซ็นต์` (สร้างโดยแอดมิน) |

### รูปแบบของ `discount_codes.txt`

```
SAVE105423,10
SAVE2091,20
SAVE505678,5
```

---

> **หมายเหตุ:** หากไม่พบไฟล์ `discount_codes.txt` ระบบจะแจ้ง `Cannot open discount file` และไม่ทำการหักส่วนลด ให้แอดมินสร้างรหัสส่วนลดก่อนใช้งาน
